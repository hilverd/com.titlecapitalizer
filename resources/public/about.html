Title Capitalizer alpha

Title Capitalizer is a small web application for capitalizing titles according to various styles. There are plenty of resources describing how one should capitalize words in a title, but there do not appear to exist many sophisticated tools for doing so automatically. This is my attempt at an implementation that uses part-of-speech tagging and allows for different capitalization styles to be specified. A style is a list of rules stating e.g. that the first and last words should always be capitalized, that prepositions of four or more letters should be capitalized, and so on.

The idea is simple: you provide a title (in lower case), which is analyzed by a part-of-speech tagger to determine which words are nouns, verbs, etc. Then the chosen capitalization style is applied, by trying to match each word to a rule and capitalizing the word if necessary. The basic architecture of Title Capitalizer is described semi-formally in Capitalization Styles.

Please note that this is only a first version and Title Capitalizer should currently be considered alpha quality. It is implemented using Clojure, ClojureScript, Twitter Bootstrap, and jQuery, and deployed to Heroku.
